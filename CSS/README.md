## Project Overview

The Second Chapter is a non-profit organisation focused on promoting literacy through free community book exchanges and reading programmes in under-resourced South African communities.

**Mission:** To make reading accessible to every child and adult by creating free community book exchanges and literacy programmes.

**Vision:** A South Africa where no community is a book desert, and every person has access to stories that inspire, educate, and empower.

## Responsive Design Breakpoints

| Breakpoint | Device | Layout Changes |
|------------|--------|----------------|
| 1200px+ | Desktop | 3-4 columns, full navigation |
| 769px - 1023px | Tablet | 2 columns, smaller padding |
| 481px - 768px | Mobile | Single column, hamburger menu |
| 480px and below | Small Mobile | Single column, smaller fonts |
